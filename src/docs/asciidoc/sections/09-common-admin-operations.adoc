== Common Admin Operations

=== Protocol Constraints (Pre-validation)

Use `ProtocolConstraints` to validate user input before sending requests.
This is recommended for settings UIs to fail fast and avoid avoidable radio round trips.

[source,java]
----
// Throwing validation.
ProtocolConstraints.validateChannelIndex(slot);
ProtocolConstraints.validateChannelName(channelName);
ProtocolConstraints.validateChannelPsk(pskByteString);

// Non-throwing validation for form-level error display.
List<ProtocolConstraints.ValidationIssue> issues =
        ProtocolConstraints.validateChannelIssues(channelPayload);
if (!issues.isEmpty()) {
    renderValidationErrors(issues);
}
----

=== Channel Helpers

[source,java]
----
admin.setChannelName(2, "meshMKT", AdminWriteMode.VERIFY_APPLIED);

// Password/PSK update
admin.setChannelPassword(2, "shared-secret", AdminWriteMode.VERIFY_APPLIED);
admin.setPrimaryChannelPassword("primary-secret", AdminWriteMode.VERIFY_APPLIED);
----

=== Security + MQTT Helpers

[source,java]
----
Config.SecurityConfig security = Config.SecurityConfig.newBuilder()
        .setAdminChannelEnabled(true)
        .build();
Config securityPayload = Config.newBuilder().setSecurity(security).build();
admin.setConfig(securityPayload, AdminWriteMode.VERIFY_APPLIED);

ModuleConfig.MQTTConfig mqtt = ModuleConfig.MQTTConfig.newBuilder()
        .setEnabled(true)
        .setAddress("mqtt.example.com")
        .setUsername("mesh-user")
        .build();
ModuleConfig mqttPayload = ModuleConfig.newBuilder().setMqtt(mqtt).build();
admin.setModuleConfig(mqttPayload, AdminWriteMode.VERIFY_APPLIED);
----

=== Structured Result Variants

All canonical write paths already return `AdminWriteResult`:

[source,java]
----
admin.setChannel(2, channel, AdminWriteMode.VERIFY_APPLIED);
admin.setConfig(config, AdminWriteMode.VERIFY_APPLIED);
admin.setModuleConfig(moduleConfig, AdminWriteMode.VERIFY_APPLIED);
admin.setOwner(targetNodeId, "Long Name", "short", AdminWriteMode.VERIFY_APPLIED);
----
